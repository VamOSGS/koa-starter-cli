# koa-starter-cli

### Usage

### Installing:

```bash
  npm install -g koa-starter-cli
```

or

```bash
yarn global add koa-starter-cli
```

### Generate project:

```bash
koa-starter <PROJECT_NAME>
```

### Commands:

* use yarn

```bash
koa-starter --use-yarn
```

* help

```bash
koa-starter -h
```

```bash
koa-starter -help
```

* version

```bash
koa-starter -v
```

```bash
koa-starter -version
```
